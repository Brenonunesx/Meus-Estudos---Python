# Meu Espaço de Estudos em Python

Bem-vindo(a) ao meu repositório de aprendizado!Criei esse espaço para centralizar e organizar minha tagetoria no aprendizado em Python, inclui minhas resoluções de exercícios, desafios de lógica, algoritmos e atividades acadêmicas.

##  Objetivos do Repositório
- Praticar a sintaxe e os conceitos da linguagem Python.
- Desenvolver e aprimorar o raciocínio lógico e a eficiência de algoritmos.
- Documentar a minha evolução na programação na minha carreira
- Manter um histórico organizado de atividades práticas para usar de revisão.

## Estrutura do Repositório
organizei as pastas dessa forma aqui:
- `exercicios/`: Questões de plataformas de desafios e testes de lógica.
- `atividades/`: Scripts,codigos e algoritmos feitos no horario academico.
- `desafios/`: Algoritmos mais complexos e codigos um pouco mais longos.

##  Tecnologias e Ferramentas
- **Linguagem:** Python 3.x
- **Ambiente de Desenvolvimento:** VS Code.
