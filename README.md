# 🐍 Meu Espaço de Estudos em Python

Bem-vindo(a) ao meu repositório de aprendizado! Este espaço foi criado para centralizar e organizar toda a minha jornada de estudos em Python, incluindo resoluções de exercícios, desafios de lógica, algoritmos e atividades acadêmicas.

## 🎯 Objetivos do Repositório
* Praticar a sintaxe e os conceitos da linguagem Python.
* Desenvolver e aprimorar o raciocínio lógico e a eficiência de algoritmos.
* Documentar a minha evolução na programação na minha carreira em ciência de dados.
* Manter um histórico organizado de atividades práticas.

## 📂 Estrutura do Repositório
O projeto está organizado da seguinte forma:
* `exercicios/`: Questões de plataformas de desafios e testes de lógica.
* `atividades/`: Scripts e projetos práticos desenvolvidos para aulas e estudos autônomos.
* `desafios/`: Algoritmos mais complexos e maratonas de código.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **Ambiente de Desenvolvimento:** VS Code.
